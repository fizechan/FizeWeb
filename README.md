# FizeWeb
Web环境参数变量的获取和设置类库。

FizeWeb 提供诸如：Cookie 、Request 、Response 、Session 等 WEB 相关的变量参数的设置、获取、处理等操作。

FizeWeb 暴露的接口简单明了，易于使用。

FizeWeb 有非常完善的参考文档。
